Learn JavaScript: Practical Tutorials for Beginners - Recording Audio

Just doing it is a way to learn. This exercise starts with Javascript (code) and stays there, presenting a simple webapp with interactive code that you can run and modify from a browser. It begins with a simple voice recorder, adding additional features like cloud API's in subsequent lessons. Simple javascript code using an interactive approach: running, editing, and rerunning code is a great way to learn. In lesson # 1, See how less than 15 lines of javascript provide an audio recorder. Understand the foundation of a small js app for media recording. Follow links in lesson sections to drill down further.
